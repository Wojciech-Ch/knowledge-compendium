wyróżniamy wiele metod których możemy użyć na arrayach
  • push dodaje wartość na koniec tablicy.
    • let arr = [1,2,3,4];
    arr.push(5);
    output: [1,2,3,4,5]
  • unshiftdodaje wartość na początek tablicy.
    • let arr = [1,2,3,4];
    arr.unshift(5);
    Output: [5,1,2,3,4,5]
  • pop
    • let arr2 = [10,20,30];
    arr2.pop()
    output: [10,20]
