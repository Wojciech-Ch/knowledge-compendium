Data Binding w Angularze - Podstawy i Przykłady

Interpolacja

Interpolacja to najprostsza forma data bindingu. Umożliwia ona wyświetlanie wartości zmiennych z komponentu w HTML.

Property binding

Property Binding pozwala na ustawienie wartości właściwości elementu HTML lub dyrektywy.


Event Binding

Event Binding umożliwia reagowanie na różne zdarzenia, takie jak kliknięcia czy wprowadzanie danych.


Two-Way Binding

Two-Way Binding łączy Property i Event Binding, umożliwiając dwukierunkową synchronizację danych między modelem a widokiem.
